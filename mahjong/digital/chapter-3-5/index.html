First, passing the first call. In the situation above, we have a eyo 3-sided shape, and two yakuhai pairs in 35. It looks like we can get into tenpai very soon. The 3 comes out. Calling gives us a really good 2000-point tenpai. What would our hand be if we didn't call? What's the difference compared to last section's hand E?

The key is we are early on in a round, and have a wide 1-shanten that looks like we can declare riichi any moment. We also have two yakuhai pairs. Even if we pass up one tile, we still have three tiles we can potentially call.

As a result passing up on the first tile is best here. The hand is still fast if we wait for the next tile to call, so we can keep your hand closed for now to aim for a more valuable hand.

Now look at this. 1-shanten for tanyao-pinfu-sanshoku. If we stay closed and riichi, this is an important hand that can easily become a mangan or haneman. If we call here, we will only have tanyao-sanshoku, a 2-han 2000-point hand. This is just a rough guideline, but when calling decreases the size of a hand by 3 han or more, we should probably stay closed to make our hand big, even if we have to make our hand slower as a result.
